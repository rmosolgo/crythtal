# Crythtal

A lisp in Crystal, just for fun.

Cues taken from:
  - [Lispy tutorial](http://norvig.com/lispy.html), building a lisp in Python
  - [Crisp](https://github.com/rhysd/Crisp), another lisp in Crystal
